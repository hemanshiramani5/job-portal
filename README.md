# job-portal
JobWave is an online job portal designed to bridge the gap between job seekers and employers through a modern, user-friendly platform.
It offers powerful search features such as real-time autocomplete and filters to simplify the job search process. 
Job seekers can build interactive resumes with customizable templates, live previews, and PDF export options,
while employers gain tools to post jobs, manage applications, and review candidate profiles effectively. The platform also supports real-time notifications, 
messaging, and an admin panel for managing templates and users. 
By integrating these features, JobWave enhances recruitment efficiency and empowers users to build professional careers seamlessly.
